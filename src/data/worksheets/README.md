# Cards Against Bureaucracy — Worksheet Pack
> 8 markdown templates for structured session capture

---

## How to use

Each file is a standalone template for one phase of a Cards Against Bureaucracy session. Work through them in order, or jump to the sheet that matches your current phase.

Fill in the blanks, check the boxes, and use the tables to capture decisions and actions. Everything is designed to be completed in the session, not after.

## Worksheets

| # | File | When to use |
|---|---|---|
| 1 | `01-session-log.md` | Start of every session — mission, team, context |
| 2 | `02-five-card-spread.md` | After drawing cards — capture the spread + sensemaking |
| 3 | `03-change-formula-check.md` | When using the Change Formula card — readiness audit |
| 4 | `04-decide-primary-picks.md` | Decision phase — choose your primary barrier + enabler |
| 5 | `05-deploy-72-hour-thin-slice.md` | Planning phase — design and commit to your experiment |

| 7 | `07-measures-and-learning.md` | Before deploying — define your signals and learning loop |
| 8 | `08-after-action-review.md` | After check-in — honest debrief and next decision |
